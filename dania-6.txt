Readme yang sekarang mencakup instruksi tugas A dan tugas B, seperti berikut:
# Pelatihan 1 Frontend OmahTI 2026 - Git & GitHub
### Instruksi
#### Tugas A:
1. Terdapat branch 'soal', pull branch tersebut dan kerjakan soal sesuai dengan instruksi yang ada pada branch tersebut
2. Kerjakan soal tersebut di branch soal dan namakan file 'namapanggilan-nomorsoal.txt'
3. Merge branch tersebut ke main

#### Tugas B:
1. Buat branch baru dengan nama 'namapanggilan'
2. Masukkan 1 file random, format dan isi dibebaskan asal tidak mengandung data pribadi atau SARA.
3. Buat pull request untuk branch tersebut ke main.
